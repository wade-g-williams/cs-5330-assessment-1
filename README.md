## Reflection

Teaching this topic forced me to think deeply about how to do convolution and padding across various applications. It made me realize that there are multiple ways to pad and do convolution in the same application and other applications warrant a particular one. I was surprised at how difficult it was to find good sources on this topic. I expected it to be a simple answer, but instead, it ended up being a highly grey area with a lot of subjectivity, nuance, and variability across sources. Many sources even contained what I think is incorrect information. This project certainly made me appreciate the nuance and complexity involved in image processing pipelines across various applications and the value that design decisions bring to engineering effective image processing pipelines.
